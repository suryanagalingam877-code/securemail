---

## Installation

### Windows

```bash
git clone https://github.com/suryanagalingam877-code/securemail.git

cd securemail

pip install -r requirements.txt

python app.py
```

---

### Linux

```bash
sudo apt update

git clone https://github.com/suryanagalingam877-code/securemail.git

cd securemail

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python3 app.py
```
